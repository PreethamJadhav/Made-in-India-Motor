# Made-in-India-Motor
Responsive car &amp; bike showcase website built with HTML, CSS &amp; JavaScript — featuring brand line-ups, model specs, and an interactive detail view.
Made in India Motors — Cars & Bikes Showcase
A responsive showcase website for Indian car and motorcycle manufacturers, built to practice dynamic UI rendering, interactive modals, and real-world front-end data structuring.

🔗 Live Demo: https://preethamjadhav.github.io/Made-in-India-Motor/

Features

* Fully responsive layout across mobile, tablet, and desktop using CSS Grid & Flexbox
* Cars / Bikes toggle switch with dynamic panel switching
* Brand cards for major Indian manufacturers (Tata, Mahindra, Maruti Suzuki, Bajaj, TVS, Royal Enfield, and more)
* Interactive modal system — Brand → Model List → Full Model Detail (engine, transmission, mileage/range, pricing, variants)
* Custom SVG icon system representing vehicle types (hatchback, SUV, sedan, cruiser, sport, scooter, adventure, electric)
* Heritage timeline section and live-counted brand/model statistics
* Smooth hover states and transitions on cards and buttons

Tech Stack

* HTML5
* CSS3 (custom properties / variables for consistent theming)
* Vanilla JavaScript (DOM manipulation, dynamic rendering, no framework)
* Google Fonts (Oswald & Inter)

What I Learned

* Structuring and rendering data-driven UI using JavaScript objects and arrays
* Building a multi-level modal navigation flow (list → detail → back) without a framework
* Writing reusable functions to avoid duplicating rendering logic across sections
* Designing a consistent dark theme using CSS variables
* Deploying a static site using GitHub Pages
